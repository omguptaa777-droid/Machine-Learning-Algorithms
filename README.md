# Machine Learning Algorithms from Scratch

A collection of machine learning algorithms implemented from scratch using **Python** and **NumPy** to develop a deeper understanding of the mathematical foundations behind machine learning. This repository focuses on implementing the core logic of each algorithm without relying on scikit-learn during development, followed by validation against the corresponding scikit-learn implementation.

---

## Overview

The goal of this project is to bridge the gap between the mathematical concepts of machine learning and their practical implementation.

Each algorithm is built from first principles, covering the complete workflow from data preprocessing and model training to prediction and evaluation.

---

## Algorithms Implemented

### Supervised Learning

- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)

### Unsupervised Learning

- Principal Component Analysis (PCA)

### Optimization

- Stochastic Gradient Descent (SGD)

Each implementation is validated by comparing its output with the equivalent implementation available in **scikit-learn**.

Validation includes:

- Predictions
- Model coefficients
- Classification accuracy
- Regression performance
- PCA transformations
- Consistency of results

## Motivation

Understanding how machine learning algorithms work internally is essential for building intuition beyond using high-level libraries.

Rather than treating machine learning models as black boxes, this project focuses on implementing the underlying mathematics and algorithms from scratch, helping build a solid foundation for advanced machine learning and deep learning.
